## Referência de comandos rápidos para trabalhar localmente

Uma rápida referência dos comandos que você vai precisar quando trabalhar localmente.

| comando | descrição |
| ------- | ----------- |
| `npm run bootstrap` | Inicializa os diferentes serviços |
| `npm run seed` | Faz o <i>parse</i> dos arquivos markdown de desafios e os insere no MongoDB. |
| `npm run develop` | Inicia o Servidor de API do freeCodeCamp e as Aplicações Clientes. |
| `npm test` |  Executa todos os testes JS no sistema, incluindo os testes de cliente, servidor, lint e testes dos desafios. |
| `npm run test:client` | Executa os testes do cliente. |
| `npm run test:curriculum` | Executa os testes do currículo. |
| `npm run test:server` | Executa os Testes do servidor. |
| `npm run commit` | Uma ferramenta interativa que te ajuda a criar uma boa mensagem de commit. |
