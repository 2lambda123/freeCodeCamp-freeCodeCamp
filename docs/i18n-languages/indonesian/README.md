## Codebase dan kurikulum open-source milik freeCodeCamp.org's

[freeCodeCamp.org](https://www.freecodecamp.org) adalah komunitas yang ramah di mana Anda dapat belajar kode secara gratis. Ini dijalankan oleh [nirlaba 501(c)(3) yang didukung donor](https://donate.freecodecamp.org) untuk membantu jutaan orang dewasa sibuk untuk bertransisi ke pekerjaan di bidang teknologi. Komunitas kami telah membantu lebih dari 10.000 orang mendapatkan pekerjaan developer pertama mereka.

Kurikulum pengembangan web full-stack kami sepenuhnya gratis dan serba mandiri. Kami memiliki ribuan tantangan kode interaktif untuk membantu Anda mengembangkan keterampilan Anda.

### Sertifikasi

freeCodeCamp.org menawarkan beberapa sertifikasi developer gratis. Masing-masing sertifikasi ini melibatkan pembangunan 5 proyek aplikasi web yang diperlukan, bersama dengan ratusan tantangan kode opsional untuk membantu Anda mempersiapkan proyek-proyek tersebut. Kami memperkirakan bahwa setiap sertifikasi akan membutuhkan programmer pemula sekitar 300 jam untuk diperolehkan.

Masing-masing dari 30 proyek ini dalam kurikulum freeCodeCamp.org memiliki user story yang bersifat agile dan tes terotomasi. Ini membantu Anda membangun proyek Anda secara bertahap dan memastikan Anda telah memenuhi semua user story sebelum Anda mengirimkannya.

Anda dapat pull suite tes ini melalui [CDN freeCodeCamp](https://cdn.freecodecamp.org/testable-projects-fcc/v1/bundle.js). Ini berarti Anda dapat membangun proyek ini di situs web seperti CodePen dan Glitch - atau bahkan di lingkungan pengembangan komputer lokal Anda.

Setelah memperoleh sertifikasi, Anda akan selalu memilikinya. Anda akan selalu dapat menautkannya dari LinkedIn atau resume Anda. Dan ketika calon atasan Anda atau klien freelance mengklik tautan itu, mereka akan melihat sertifikasi terverifikasi yang khusus untuk Anda.

Satu-satunya pengecualian adalah jika kami menemukan pelanggaran terhadap [Kebijakan Kejujuran Akademik](https://www.freecodecamp.org/academic-honesty) kami. Ketika kita menangkap orang menjiplak dengan jelas (menyerahkan kode atau proyek orang lain sebagai milik mereka tanpa kutipan), kita melakukan apa yang harus dilakukan oleh semua institusi pembelajaran yang ketat - kita mencabut sertifikasi mereka dan melakukan ban terhadap orang-orang itu.

Kode ini sedang berjalan live di [freeCodeCamp.org](https://www.freecodecamp.org).

Komunitas kami juga punya:

### Melaporkan Bug dan Isu
Jika Anda merasa telah menemukan bug, pertama baca artikel [bagaimana melaporkan bug](https://www.freecodecamp.org/forum/t/how-to-report-a-bug/19543) dan ikuti instruksinya.

Jika Anda yakin ini adalah bug baru dan telah mengkonfirmasi bahwa orang lain menghadapi masalah yang sama, silakan dan buat issue GitHub baru. Pastikan untuk memasukkan informasi sebanyak mungkin agar kami dapat mereproduksi bug tersebut.

### Melaporkan Masalah Keamanan

Jika Anda merasa telah menemukan kelemahan sekuritas, harap laporkan secara bertanggung jawab. Jangan membuat issue GitHub untuk masalah keamanan. Sebagai gantinya, silakan kirim email ke `security@freecodecamp.org` dan kami akan segera memeriksanya.
