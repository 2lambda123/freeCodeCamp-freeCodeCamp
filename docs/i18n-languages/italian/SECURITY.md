## Riportare una vulnerabilita'

I prpblem di vulnerabilita' dovrebbero essere inviati per email a  security@freecodecamp.org, per favore non creare un GitHub issue pubblico.

Assicurati che tu stia usand la version **latest**, **stable** and **updated** del tuo sistema operativo e del browser disponibil sul tuo computer.
