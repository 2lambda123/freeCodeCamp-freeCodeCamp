## Comandi per lavorare localmente

Ecco una lista veloce di comandi che ti serviranno quando vuoi lavorare localmente.

| commando | descrizione |
| ------- | ----------- |
| `npm run bootstrap` | Inizializza i vari servizi. |
| `npm run seed` | Parsing di tutti i file challenge markdown e inserimento in MongoDB                                                                      |
| `npm run develop` | Lancia l'API del Server di freeCodeCamp e le applicazioni Client.                                                           |
| `npm test` |  Lancia tutti i test in JavaScript presenti.                                                                                   |
| `npm run test:client` | Lancia la suite per il test. |
| `npm run test:curriculum` | Lancia la suite per il curriculum. |
| `npm run test:server` | Lancia la suite per il server. |
| `npm run commit` | Un tool interattivo per aiutarti a costruire un buon messaggio di commit.                                                            |
