# Hızlı referans dokümantasyonu

Merhaba 👋!

Bu rehber freeCodeCamp.org'e katılan bütün dokümantasyonu içermektedir.

## [Eğer yeni başlıyorsanız, bunu okuyarak başlayın.](/CONTRIBUTING.md)

---

## Hızlı referans makaleleri

<a href="">1. Rehber makaleler nasıl çalışır.</a><br>
<a href="">2. Kodlama meydan okumaları nasıl çalışır.</a><br>
<a href="">3. freeCodeCamp'ı ücretsiz nasıl yüklerim.</a><br>
<a href="">4. Yerel olarak giden emailleri nasıl yakalarız.</a><br>

## Stil rehberi

1. Kılavuz makale oluşturmak için stil rehberi.
2. Kodlama meydan okuması oluşturmak için stil rehberi.

## Local'de çalışırken, hızlı komut referansları

Local'de çalışırken ihtiyaç duyacağın komutların hızlı referansı

| komut | açıklama |
| ------- | ----------- |
| `npm run bootstrap` | Farklı servisleri yükler.  | 
| `npm run seed` | Meydan okuma dosyalarını ayırır ve bunları MongoDB'ye ekler. |
| `npm run develop` |  | freeCodeCamp API Server'ı ve Müşteri Uygulamaları'nı başlatır. |
| `npm test` |  Sistemdeki bütün JS testlerini,client,server,meydan okuma testleri, başlatır. |
| `npm run test:client` | Kullanıcı test ortamını başlatır. |
| `npm run test:curriculum` | Eğtiim test ortamını başlatır. |
| `npm run test:server` | Server test ortamını başlatır. |
| `npm run commit` | Bir iş'in mesajını oluşturmak için ihtiyaç duyacağın aktif eklentidir. |
