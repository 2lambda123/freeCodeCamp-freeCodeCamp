##快速命令在本地工作時引用

在本地工作時快速參考您需要的命令。

|命令|描述|
| ------- | -----------
| `npm run bootstrap` |引導不同的服務|
| `npm run seed` |解析所有質詢標記檔並將其插入MongoDB。 |
| `npm run develop` |啟動freeCodeCamp API伺服器和用戶端應用程式|
| `npm test` |在系統中運行所有JS測試，包括用戶端，伺服器，lint和質詢測試
| `npm run test：client` |運行用戶端測試套件|
| `npm run test：courses` |運行課程測試套件|
| `npm run test：server` |運行伺服器測試套件|
| `npm run commit` |一個説明您構建良好提交消息的互動式工具
