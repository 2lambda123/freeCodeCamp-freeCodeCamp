### --question--

#### --text--

How would you calculate the nth number in the Fibonacci sequence? The Fibonacci sequence starts as `0, 1, 1, 2, 3, 5, 8...`

#### --distractors--

Use the formula `fib(n) = n * (n-1)`.

---

Use recursion with memoization to avoid recalculating subproblems where `fib(n) = n + (n-1)`.

---

Use a for loop to calculate iteratively.

#### --answer--

Use recursion where `fib(n) = fib(n-1) + fib(n-2)`.

### --question--

#### --text--

How would you convert a binary string (e.g., "1011") to its decimal equivalent?

#### --distractors--

Multiply each bit by powers of 2 and sum them up.

---

Convert the string to an array, then sum the digits using bitwise operations.

---

Use the `Number()` function directly with a binary string.

#### --answer--

Use the `parseInt()` function with base 2.

### --question--

#### --text--

Which of the following algorithm will move all zeroes in an array to the end while keeping the order of other elements unchanged. For example, given `[0, 1, 0, 3, 12]`, return `[1, 3, 12, 0, 0]`.

#### --distractors--

Sort the array in decreasing order.

---

Sort the array and place zeros at the end.

---

Use recursion to remove all zeros and add them to the end.

#### --answer--

Use a two-pointer technique to swap non-zero elements with zero elements.

### --question--

#### --text--

How would you reverse the order of words in a sentence? For example, given the input `"I am a frontend engineer"`, the output should be `"engineer frontend a am I"`.

#### --distractors--

Split the string into words using `split(" ")`, reverse the array, and join it back into a string using `join("_")`.

---

Sort the words alphabetically and return the result.

---

Use regular expressions to reverse each word.

#### --answer--

Split the string into words using `split(" ")`, reverse the array, and join it back into a string using `join(" ")`.

### --question--

#### --text--

Given a string, what would be the best approach to find the first non-repeating character. For example, in the string "codecamp", it would be "o".

#### --distractors--

Sort the string and check for consecutive duplicates.

---

Convert the string to an array, then filter for unique elements.

---

Use a regular expression to find the first non-repeat.

#### --answer--

Iterate over the string twice, once to count number of occurrences and once to find the first unique character.
