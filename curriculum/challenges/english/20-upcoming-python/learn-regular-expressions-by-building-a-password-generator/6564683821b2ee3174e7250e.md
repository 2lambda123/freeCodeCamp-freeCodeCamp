# --description--

You can access to the utilities defined inside the imported module through the dot notation, appending a dot followed by the utility name to the module name. For example:

```py
import string

print(string.ascii_lowercase) 

# Output: abcdefghijklmnopqrstuvwxyz
```

In this project, you are going to use different constants from the `string` module. 

Create a new variable called `letters` and assign it `string.ascii_letters`.
