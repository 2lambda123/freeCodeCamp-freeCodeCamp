We've already created an `if` statement that checks if the `left_part` index is less than the `right_part` index. 

Using an `else` statement will execute code when `left_part` index is not less than the `right_part` index. So the `else` statement will only run when the `right_part` index is less than the `left_part` index.

Create an `else` statement. In its code block/suite, assign `right_part[right_array_index]` to `array[sorted_index]`, and increment the `right_array_index` by 1.

    right_part = array[middle_point:]

    merge_sort(right_part)

            array[sorted_index] = left_part[left_array_index]
