# --description--

You want to keep track of the paths between the starting node and each other node too. You can create a dictionary with key-value pairs in the form of node-list_of_nodes. For that, create a `paths` variable and use a dict comprehension to build your dictionary.

```py
{key: val for key in dict}
```

This time you don't need the conditionals. Simply assign an empty list for each node in the graph.
