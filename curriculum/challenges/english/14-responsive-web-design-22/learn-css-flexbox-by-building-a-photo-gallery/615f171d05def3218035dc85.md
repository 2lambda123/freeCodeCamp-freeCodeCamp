# --description--

Right now, your images are too big. To solve this, we need to set the `box-sizing` propety to the `border-box` value. Unlike the `content-box` value, `border-box` keeps the width and height of an element remains the same and changes its padding and borders.    With this in mind, create a `.gallery img` selector to target the images that have the `.gallery` class.  Give them all a `width` of `100%` and a `max-width` of `350px` and set the `box-sizing` to `content-box` so they shrink as needed but don't get too big. Also set the `height` property to `300px` to keep your images a uniform size.
