# --description--

Before you start sizing and positioning things, you need to address this question, "how is the size of my container being calculated?" There are two different ways to do this, and they can significantly impact how our content is displayed. 

Now remember, a block-level container consists of four parts: the content of the container itself, the padding around that content, the border around that padding and the margin around the border.  With that  in mind, this is where you meet the CSS property, `box-sizing`. It lets you specify how those parts are used in calculating a container's size.
If you do `box-sizing: content-box;`, you pretty much said, "the size of the box is solely the size of its content." Any padding (the space between the content and its border) and the border are not used in calculating the size of a box. This is the default for most browsers, but let's set it explicitly.
