# --description--

Let's talk of encapsulation and getters and setters. You have written the three attributes of the class to be private using the two underscores in front. The attributes are hidden and can't be accessed by the outside.

But it may be that you want to see or update the values of the attributes of an already existing instance: in this case you use getters to see the values of the attributes and getters to set the new values.

In this step you will write the getters of the three attributes.

An example of how to write a getter is:

```py
class Nest:
    ...
    @property
    def number_of_eggs(self):
        return self.__number_of_eggs
```

This `number_of_eggs` getter returns the value of the private property `__number_of_eggs`.

To define a getter, you define a method and give it a `@property` decorator. The decorator makes so that the method is not called like a method, but used like a property.

```py
n = Nest()
print(n.number_of_eggs)
```

You can access the number of eggs in the nest simply with `n.number_of_eggs`.

Create the three getters for the three attributes. Remember that the angle is stored internally in radians, but getter should return a degrees value (`math.degrees()`, together with the `round` function could be useful).

Give the getters the same name as the private property but without the two underscores in front.

import math

GRAVITATIONAL_ACCELERATION = 9.81

class Projectile:
    __slots__ = ('__height', '__speed', '__angle')

    def __init__(self, speed, height, angle):
        self.__height = height
        self.__speed = speed
        self.__angle = math.radians(angle)
        
    def __str__(self):
        return f'''
Projectile details:
height: {self.__height} m
speed: {self.__speed} m/s
angle: {round(math.degrees(self.__angle))}Â°
displacement: {round(self.__calculate_displacement(), 1)} m
'''

    def __calculate_displacement(self):
        horizontal_component = self.__speed * math.cos(self.__angle)
        vertical_component = self.__speed * math.sin(self.__angle)
        squared_component = vertical_component**2
        gh_component = 2 * GRAVITATIONAL_ACCELERATION * self.__height
        sqrt_component = math.sqrt(squared_component + gh_component)
        
        displacement = horizontal_component * (vertical_component + sqrt_component) / GRAVITATIONAL_ACCELERATION
        return displacement
        
    def __calculate_y_coordinate(self, x):
        height_component = self.__height
        angle_component = math.tan(self.__angle) * x
        acceleration_component = GRAVITATIONAL_ACCELERATION * x ** 2 / (
                2 * self.__speed ** 2 * math.cos(self.__angle) ** 2)
        y_coordinate = height_component + angle_component - acceleration_component

        return y_coordinate
    
    def calculate_all_coordinates(self):
        return [
            (x, self.__calculate_y_coordinate(x))
            for x in range(math.ceil(self.__calculate_displacement()))
        ]
