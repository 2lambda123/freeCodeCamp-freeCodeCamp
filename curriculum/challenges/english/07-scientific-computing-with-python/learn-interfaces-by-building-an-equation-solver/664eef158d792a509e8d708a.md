# --description--

The `sub` function from the `re` module enables you to replace text inside a string based on a regex pattern.

```py
spam = 'Always look on the bright side of life'
ham = re.sub(r'bright', 'light', spam)
ham == 'Always look on the light side of life' # True
```

It takes three arguments: the regex pattern to substitute, the replacement, and the string.

From your `__str__` function, return a `sub()` call passing the string `'1'`, an empty string, and your existing `equation_string.strip('+')` call as the arguments. You'll refine the regex pattern in the next steps.
