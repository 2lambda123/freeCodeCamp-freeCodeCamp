# --description--

If the user decides not to cancel and choose `exit`, you need to hide the form modal.

To do that, add a click event listener to the `confirmBtn` element. For what happens when the user clicks the `confirmBtn`, set the display property of the form modal element to `none`.

Do not use implicit return here because you will still have something to do inside the event listener.
