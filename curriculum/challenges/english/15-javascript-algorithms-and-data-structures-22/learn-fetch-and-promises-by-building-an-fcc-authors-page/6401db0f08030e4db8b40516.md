# --description--

Now, your FCC Authors Page app is complete. But you need to clean up some things.

After clicking the "Load More Authors" button a couple of times, you'll see it doesn't fetch more authors anymore. That's because you've reached the end of the authors list. For a better UX, that button should be disabled at that point and the text set to something like "No more data to load". You can also set the cursor to `not-allowed` to make sure the user stop clicking it. 

If you look at the authors' bios too, some are longer than the others. you can extract the part of the bios you want and replace the others with an ellipsis (...).

Also, to make things look better you can add a divider line between the authors' images and every other thing in the authors cards.

First, let's start with the "Load More Authors" button. When there's no more data to load, you want to change the button text and disable the button. An `if` statement would be ideal for this.

Inside the `fetchMoreAuthors` function, write an `if` statement and set the condition to `authorDataArr.length <= endingIndex` â€“ meaning there's no more data to load.

const authorContainer = document.getElementById('author-container');
const loadMoreBtn = document.getElementById('load-more-btn');

let startingIndex = 0;
let endingIndex = 8;
let authorDataArr = [];

fetch('https://cdn.freecodecamp.org/curriculum/news-author-page/authors.json')
  .then((res) => res.json())
  .then((data) => {
    authorDataArr = data;
    displayAuthors(authorDataArr.slice(startingIndex, endingIndex));  
  })
  .catch((err) => {
    console.error(`There was an error: ${err}`);
  });

const fetchMoreAuthors = () => {
  startingIndex += 8;
  endingIndex += 8;

  displayAuthors(authorDataArr.slice(startingIndex, endingIndex));

  --fcc-editable-region--
};

const displayAuthors = (authors) => {
  authors.forEach( ({ author, image, url, bio }, index) =>
  (authorContainer.innerHTML += `
    <div id="${index}" class="user-card">
      <h2 class="author-name">${author}</h2>
      <img class="user-img" src="${image}" alt="${author} avatar">
      <p class="bio">${bio}</p>
      <a class="author-link" href="${url}" target="_blank">${author} author page</a>
    </div>
  `)
  )
};

loadMoreBtn.addEventListener('click', fetchMoreAuthors);
