# --description--

When you make a `GET` request to an endpoint, the data you get is not always usable at first. To make it usable, you can use the `json()` method on the response with the `.then()` method.

The `.then()` method can take a callback function for the fulfilled and rejected cases of the `Promise`.

Add a `.then()` method right under your `fetch()`. Inside the `.then()` method, name the response from the endpoint `res`, add a fat arrow for the callback, then use the `json()` method on that `res`. You don't need to use curly braces because the arrow function will have one expression. 

Again, don't terminate the code with a semi-colon yet.
