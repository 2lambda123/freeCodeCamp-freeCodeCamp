# --description--

Apart from the `.then()` method, the `.catch()` method is another asynchronous JavaScript method with which you can handle errors. This is useful in case the `Promise` gets rejected.

Chain a `.catch()` to the last `.then()`. Set the error variable to `err` inside the `catch()`. Add your callback, then using `console.error()`, log that error to the console with the text `There was an error: ${err}`. Since you're using the `err` variable in the text, don't forget to use template literal (\`\`) instead of single or double quotes.

**Note**: Now you can terminate the code with a semi-colon. You can't do that in the previous steps because you'll signal to JavaScript to stop parsing your code, which will affect your `fetch()` syntax.
