# --description--

Arrays are special in that they are considered <dfn>mutable</dfn>. This means you can change the value at an index directly.

For example, this code would assign the number `25` to the second index in the array:

```js
let array = [1, 2, 3];
array[1] = 25;
```

Update the value in the **third** index of your `rows` array to be the number `10`. Then print the `rows` array to your console.
