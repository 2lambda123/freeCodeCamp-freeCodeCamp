# --description--

Variables can be <dfn>assigned</dfn> a value. When you do this at the same time that you declare it, this is called <dfn>initialization</dfn>. For example:

```js
var camperbot = "Bot";
```

This would initialize the `camperbot` variable with a value of `Bot`. Initialize your `xp` variable to have a value of `0`.
