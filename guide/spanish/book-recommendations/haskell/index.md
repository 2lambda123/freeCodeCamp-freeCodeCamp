#### Programming in Haskell (Second Edition)


Disponible para comprar o descargar gratis (e-book)! Este libro se divide en dos partes; la primera parte se centra en los conceptos básicos de Haskell, mientras que la segunda parte le presenta temas avanzados. Haskell es un lenguaje de programación funcional que permite a los programadores desarrollar software claro y conciso. Este libro también incluye muchos ejemplos, ejercicios, diapositivas de conferencias descargables y código de ejemplo del sitio web de (http://www.cs.nott.ac.uk/~pszgmh/pih.html) [University of Nottingham].
