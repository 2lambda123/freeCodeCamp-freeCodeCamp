## Base Tag

This is a stub. <a href='https://github.com/freecodecamp/guides/tree/master/src/pages/html/elements/base-tag/index.md' target='_blank' rel='nofollow'>Help our community expand it</a>.
The ```<base>``` tag sets the base URL to which all relative URLs on the page refer.
The ```<base>``` tag can take two attributes: ```href``` and ```target```.  It is located in the ```<head>```.  There can only be one ```<base>``` tag in the document.
