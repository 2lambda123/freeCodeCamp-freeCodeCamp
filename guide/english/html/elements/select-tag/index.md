The ```<select>``` element is used along with one or more ```<option>``` elements to create a drop down list. The ```<select>``` element is used to create the drop-down list and the ```<option>``` tag inside of the ```<select>``` element is used to show all of the available options in the list.

Here is an example of what a select dropdown looks like:

![Dropdown](https://i.imgur.com/glzQeFu.jpg)

To create one yourself, follow the example below:
```HTML
<!-- This select tag features 4 programming languages but you can use anything!-->
<select>
    <option value="JavaScript">JavaScript</option>
    <option value="Python">Python</option>
    <option value="Java">Java</option>
    <option value="Ruby">Ruby</option>
</select>
```
