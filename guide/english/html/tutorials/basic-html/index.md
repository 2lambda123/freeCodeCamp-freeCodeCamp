## Basic HTML

Structure of a basic HTML file:

```html
<!doctype html>
<html>

  <head>
    <title></title>
  </head>
  
  <body>
  </body>
</html>  
```


### Sample HTML Program

```html
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
```
