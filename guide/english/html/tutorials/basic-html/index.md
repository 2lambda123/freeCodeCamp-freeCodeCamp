<p>
  1) HTML stands for Hyper Text Markup Language<br>
  2) HTML elements are the building blocks of HTML pages<br>
  3) HTML elements are represented by tags<br>
  4) HTML tags label pieces of content such as "heading", "paragraph", "table", and so on<br>
  5) Browsers do not display the HTML tags, but use them to render the content of the page<br>
</p>

Structure of a basic HTML file:

```html
<!doctype html>
<html>

  <head>
    <title></title>
  </head>
  
  <body>
  </body>
</html>  
```
