# Visual Studio Code

<a href='https://github.com/Microsoft/vscode' target='_blank' rel='nofollow'>![logo](https://upload.wikimedia.org/wikipedia/commons/thumb/2/2d/Visual_Studio_Code_1.18_icon.svg/64px-Visual_Studio_Code_1.18_icon.svg.png "VS Code logo")</a>

Visual Studio Code is a source code editor by Microsoft and is distributed on Windows, Linux and macOS. 
Supports several progamming languages and is open source. A neat feature of VS Code is the command palette (similar to sublime text editor). The command palette is a command-line interface.
