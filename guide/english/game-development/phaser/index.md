## Phaser

Phaser is an open source framework for developing HTML5 games for desktop and mobile.

Games ranging from the GameBoy-like <a href='https://nettleflap.itch.io/lurk' target='_blank' rel='nofollow'>LURK</a> to the multiplayer <a href='https://coreyvh.itch.io/monarchies' target='_blank' rel='nofollow'>Monarchies of
Wax and Honey</a> are all created with JavaScript magic and can be played in-browser.
