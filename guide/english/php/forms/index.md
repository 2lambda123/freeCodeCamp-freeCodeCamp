## Forms

PHP forms are used to take input from vistors. When the user fills out the form above and clicks the submit button, the form data is sent for processing to a PHP file named "welcome.php". The form data is sent with the HTTP POST method.

Know more: [PHP Forms](https://www.w3schools.com/php/php_forms.asp)
