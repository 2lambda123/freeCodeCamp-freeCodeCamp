## Groovy
Apache Groovy or Groovy is a powerful and dynamic language with static compilation and typing capabilities for the Java platform and was designed to increase productivity with its concise and familiar syntax. It integrates with any Java program with ease.

### History
The first mention of Groovy was by James Strachan in a blog poast titled [Groovy - the birth of a new dynamic language for the Java platform](https://web.archive.org/web/20030901064404/http://radio.weblogs.com/0112098/2003/08/29.html). The initial Groovy release, Groovy 1.0, came out on 2 January 2007. Groovy 2.5 is the latest stable version of Groovy.
