## Bash command: chmod

**Change the permissions of a File/Directory** ,for example `chmod u+x hello.py`.(Gives the user the permission to execute the file)

### Usage

```bash
chmod [options] mode[,mode] file_name1 [file_name2 ...]
```

Most used options:

* `-R`, recursive, include all objects in subdirectories
* `-f`, force processing to continue even if errors occur
* `-v`, verbose, show all objects changed
