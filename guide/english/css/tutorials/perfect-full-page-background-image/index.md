One of the best method to get perfect full page background image is to use it in HTML tag. Below is sample code for it.

```
html {
  background: url(images/background.jpg) no-repeat center center fixed;
  background-size: cover;
  height: 100%;
  overflow: hidden;
}
```
This method works on all the major browsers. Find a codepen example in following link <a href="https://codepen.io/cybermandu/pen/OBrrOj/" target='_blank' rel='nofollow'> Perfect Full Page Background Image </a>.
