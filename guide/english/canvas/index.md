## HTML5 Canvas

Canvas is a technology introduced in HTML5 which can be accessed by the `<canvas>` tag. It allows graphics to be drawn via Javascript, and is a powerful tool for interactivity on the web.

#### More Information:

- [MDN Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)


The <canvas> element is only a container for graphics and you must use JavaScript to actually draw the graphics.
There are several methods in Canvas for adding images, drawing paths, boxes, circles, and text.
