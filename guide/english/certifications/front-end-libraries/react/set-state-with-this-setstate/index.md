## Set State with this.setState

```JSX
class MyComponent extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      name: 'Initial State'
    };
    this.handleClick = this.handleClick.bind(this);
  }
  handleClick() {
    this.setState({
      name: 'React Rocks!'
    });
  }
  render() {
    return (
      <div>
        <button onClick={this.handleClick}>Click Me</button>
        <h1>{this.state.name}</h1>
      </div>
    );
  }
};
```
### Code Explanation:
When users click the button the `handleClick()` method will be called and inside this method the data of the constuctor\`s state will be updated by `setState()` method. The `h1` tag will then be changed with the new data of the constructor\`s state.
