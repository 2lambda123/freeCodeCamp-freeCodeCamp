Don't know how to start testing? Use these Automated Accessibility Testing Tools to give you an idea how your content can reach its audience better.

1.<a href='https://tenon.io/' target='_blank' rel='nofollow'>Tenon </a>

Tenon tests in accordance to the Web Content Accessibility Guidelines (WCAG), highlighting portions of code where tests has failed. It also make recommendations and explain issues.

2.<a href='https://wave.webaim.org/' target='_blank' rel='nofollow'>Wave </a>

Developed by WebAIM, this tool makes markups on your site instead of on the code which may help you understand accessibility issues better. They also provide explanations on their markdowns.
