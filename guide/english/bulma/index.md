## Bulma

Bulma is an open source CSS framework based on flexbox and used by more than 100,000 developers.
Why Bulma?
* 100% Responsive.
* Modular.
* Modern.
* Free (Source code on Github).
* Has a growing community.
* Easy to learn.
* Quickly Customizable.

Just one command to get started:
```terminal
$ npm install bulma
```
