## Bulma

Bulma is an open source CSS framework based on flexbox and is used by more than 100,000 developers.
Why Bulma?
* 100% Responsive.
* Modular.
* Modern.
* Free (Source code on Github).
* Growing community.
* Easy to learn.
* Quick Customization.

Just one command to get started:
```terminal
$ npm install bulma
```
