## Функция Ord

`ord()` является встроенной в Python 3 функцией для преобразования строки, представляющей один символ Unicode, в целочисленное представление Unicode-кода символа.

#### Примеры:
```
>>> ord('d') 
 100 
 >>> ord('1') 
 49 
```

## функция chr

`chr()` - встроенная в Python 3 функция для преобразования целочисленного представления Unicode-кода в строку, представляющую соответствующий символ.

#### Примеры:
```
>>> chr(49) 
 '1' 
```

Следует отметить, что если целочисленное значение, переданное в `chr()`, выходит за допустимый диапазон, то будет возбуждено исключение ValueError.
```
>>> chr(-10) 
 'Traceback (most recent call last): 
  File "<pyshell#24>", line 1, in <module> 
    chr(-1) 
 ValueError: chr() arg not in range(0x110000)' 

```
