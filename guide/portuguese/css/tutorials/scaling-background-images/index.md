## Escala de imagens de fundo

Este é um esboço. [Ajude nossa comunidade a expandi-lo](https://github.com/freecodecamp/guides/tree/master/src/pages/css/tutorials/scaling-background-images/index.md) .

[Este guia de estilo rápido ajudará a garantir que sua solicitação de recebimento seja aceita](https://github.com/freecodecamp/guides/blob/master/README.md) .

```CSS
#example1 {
    background: url(mountain.jpg);
    background-repeat: no-repeat;
    background-size: auto;
}

#example2 {
    background: url(mountain.jpg);
    background-repeat: no-repeat;
    background-size: 300px 100px;
}
```
