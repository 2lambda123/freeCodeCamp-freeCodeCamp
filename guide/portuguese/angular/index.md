## Angular

O AngularJS (versões 1.x) é um Framework JavaScript de código aberto. É multi-plataforma e usado para desenvolver Single Page Applications (SPA). Implementa o padrão MVC para separar lógica, apresentação e componentes de dados. Ele também usa injeção de dependência para fazer o uso de serviços server-side em aplicações client-side.

O Angular (versões 2.x e posterior) é um Framework Typescript de código aberto para desenvolver aplicações web no front-end. Possui recursos como generics, tipagem estática e também alguns recursos do ES6.

## Histórico de versões

A Google lançou a versão inicial do AngularJS em 20 de outubro de 2010. A versão estável do AngularJS foi a 1.6.8, lançada em 18 de dezembro de 2017. O Angular 2.0 foi lançado em 22 de setembro de 2014 na conferência ng-Europe. Uma das características do Angular 2.0 é o carregamento dinâmico e a modularidade, a maioria das funcionalidades do core foi movida para módulos.

Após algumas modificações, o Angular 4.0 foi lançado em dezembro de 2016. Angular 4 é retrocompatível com o Angular 2.0. A biblioteca HttpClient é uma das características do Angular 4.0. O lançamento do Angular 5 ocorreu em 1º de novembro de 2017. O suporte para Progressive web apps foi uma das melhorias do Angular 5.0. O lançamento do Angular 6 foi em maio de 2018. A última versão estável é a [6.1.9](https://blog.angular.io/angular-v6-1-now-available-typescript-2-9-scroll-positioning-and-more-9f1c03007bb6)

**Instalação**:

Podemos adicionar o Angular referenciando diretamente a CDN disponível ou baixando o framework.

**Link para a CDN**:

AngularJS: Podemos adicionar o AngularJS (versões Angular 1.x) referenciando a CDN do Google.

```html
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.6.4/angular.min.js"></script> 
```

Download/instalação: Podemos baixar o framework via npm, Bower ou composer

**Angular 1.x**:

npm

```shell 
npm install angular
```
Then add a `<script>` to your `index.html`:

```html
<script src="/node_modules/angular/angular.js"></script>
```

bower

```shell
bower install angular
```
Then add a `<script>` to your `index.html`:

```html
<script src="/bower_components/angular/angular.js"></script>
```

Para mais informações sobre a documentação, consulte o site oficial do [AngularJS](https://docs.angularjs.org/api)

Você pode instalar o **Angular 2.x** e outras versões seguindo o passo a passo da documentação oficial do [Angular](https://angular.io/guide/quickstart)

*Nota: Por favor, consulte a documentação do Angular para obter informações atualizadas.*
